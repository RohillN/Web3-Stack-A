Get Specific Country


Overview
- Country name is case sensitive. (country name must be exactly the same as the name in the database)
- Both success and error of deletion will return an object.


GET /getcountries/<country_name> - Get a specific country on name


Example Usage
GET /getcountries/Japan
returns

[
    {   "_id": {"$oid": "5ed5bc030f7ba0588f332d6c"}, 
        "name": "Japan", 
        "data": {
                    "females_aged_15plus_employment_rate_percent": {data},
                    "population_total": {data},
                    "males_aged_15plus_employment_rate_percent": {data}
                }
    }
]




    
Errors - List All Countries

    
If no countries are in database, then return a message.
Example Usage
GET /getcountries
returns message

    "Error - no countries found. Please add a country."
