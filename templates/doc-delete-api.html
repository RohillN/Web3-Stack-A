DELETE API

    
        
Delete Specific Country

        
DELETE /getcountries/<country_name> - Delete a specific country on name

        
Overview
- Delete country on name input from form and return country name to delete.
- Country name is case sensitive. (country name must be exactly the same as the name in the database)
- On success will return object of country being deleted.
- On error return message with status.

        
Example Usage
DELETE /getcountries/Japan
returns country object begin deleted

[
    {   
        "_id": {"$oid": "5ed5bc030f7ba0588f332d6c"}, 
        "name": "Japan", 
        "data": {
                    "females_aged_15plus_employment_rate_percent": {data},
                    "population_total": {data},
                    "males_aged_15plus_employment_rate_percent": {data}
                }
    }
]
    
    

    
        
Errors - Delete Specific Country

        
Status: 404
If country being deleted does not exists in database, then return a message.
Example Usage
GET /getcountries/test
returns error message

        Error - Country does not exists in database. Please try another country.
