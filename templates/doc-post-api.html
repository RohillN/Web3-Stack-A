POST API

    
        
Post Country

        
POST /getcountries - Post data on country name

        
Overview
- Post data from a form and return country name wanting to be stored.
- If the country exists, error will occur. Returns status and error message.

        
Example Usage - New Country
POST /getcountries
Posting Data: Test
returns new country added

[
    {   
        "_id": {"$oid": "5ed5d9a20d918d5923396f5c"}, 
        "name": "Test", 
        "data": {}
    }
]




    
Errors - Post Existing Country

    
Status: 409
If country being added already exists in database, then return a message.
Example Usage
POST /getcountries
Posting Data: Japan
returns error message

    "Error - Country already exists in database. Please add another country."
