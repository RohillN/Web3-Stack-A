POST API

        
Post Country

        
POST /getcountries - Post data on country name

        
Overview
- Post data from a form and return country name wanting to be stored.
- If the country exists the input will overwrite the pre-existing country name.
- If the county has data, it will remain after the overwritten country name.

        
Example Usage - New Country
POST /getcountries
Posting Data: Test
returns new country added

[
    {   
        "_id": {"$oid": "5ed5d9a20d918d5923396f5c"}, 
        "name": "Test", 
        "data": {}
    }
]




    
Example Usage - Exsisting Country
POST /getcountries
Posting Data: Japan
returns overwritten country and data

[
    {   "_id": {"$oid": "5ed5bc030f7ba0588f332d6c"}, 
        "name": "Japan", 
        "data": {
                    "females_aged_15plus_employment_rate_percent": {data},
                    "population_total": {data},
                    "males_aged_15plus_employment_rate_percent": {data}
                }
    }
]
